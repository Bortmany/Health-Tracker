# Ads playbook — TikTok

Platform playbook for campaigns on TikTok. The approve-per-campaign contract in `ads/spend-policy.md` governs everything here; every proposal names which playbook it follows.

## Formats used

Vertical 9:16 video only, in practice — sizes, durations and file limits are the spec table in `assets/README.md`.

## Policy gotchas

- **Creative-native tone.** Ads that look like ads die on TikTok. Creatives must feel like something a person filmed — polished brand graphics that work on Meta usually flop here. Proposals for TikTok describe the native-style creative, not a repurposed poster.
- **Fast creative fatigue.** TikTok burns through creatives in days, not weeks — a proposal should include several variants up front and expect refresh proposals sooner than on other platforms.
- **AI-generated-media labeling.** TikTok requires AI-generated or significantly AI-altered content to be labeled as such. Any creative produced with generation tools carries the label — this is also on the brand-safety checklist in `assets/README.md`.

## Owner-button checklist (TikTok)

The owner, and only the owner, does these inside TikTok Ads Manager:

1. Create/confirm the ad account and payment method (row in `ads/accounts.md` first).
2. Create the campaign, ad groups and ads exactly as the approved card says, including the AI-content label where it applies.
3. Set the budget and schedule from the card — never more than the card's cap.
4. Press Publish.
5. Export results as CSV (or grant read-only API access) so agents can monitor.
6. Press every later pause, budget-change or kill button when a proposal card says it's time.

Every proposal names which playbook it follows.
