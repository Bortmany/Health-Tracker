# Weekly routine (Friday)

1. **analyst** — the Friday report (see agent file for structure): numbers, suggestions, activation switchboard. Email to the owner as a Gmail draft-or-send per current setup + archive in `drafts/reports/`.
2. **email-marketer** — if a sequence/newsletter is due, draft it.
3. **Every other Monday** (tracked in the calendar files): **strategist** produces the next 2-week calendar per active brand.
4. Monthly extras when their week arrives and they're active: competitor-watch digest, feedback-curator top-3, maintenance run, TradeOS listening digest.

The Friday report is the heartbeat: if the owner reads nothing else, that one email must carry the whole picture.
