# Daily routine (every morning)

Run order matters — later steps consume earlier output. Skip a step cleanly if its precondition is missing; note skips in the commit message.

1. **reddit-scout** — discovery sweep + reply drafts for active brands; listening-mode insights for TradeOS.
2. **copywriter** — write copy for today's calendar angles (if the current calendar has run out, note "strategist needed" instead of improvising a calendar).
3. **designer** — graphics for anything the copywriter left in Drafted.
4. **publisher** — act on owner-Approved cards (or prepare-only mode pre-setup).
5. **Queue hygiene** — anything in Awaiting approval older than 5 days gets one gentle nudge line in the next Thursday report, never a direct ping. If the queue has >10 waiting cards, draft FEWER tomorrow — the system adapts to the owner's pace, not the reverse.
6. Commit: one line, plain English ("Daily: 2 Cut posts drafted, 3 Reddit opportunities, 1 published").

Whole run should stay small — this is a maintenance rhythm, not a daily reinvention. If any live-check agent is active (ops-watchdog, support-agent), they run on their own schedules, not in this batch.
