Cut design system
Cut — dark, lime, calm

The redesign keeps Cut's identity — near-black canvas, the lime accent nobody else in the category owns, mono micro-labels — and matures it: real semantic colors, a spacing and type scale, shared components, and a tone that never shames. Direction set by competitor research across MyFitnessPal, MacroFactor, Hevy, Strong, Fitbod, Noom, Caliber and Trainerize (docs/competitor-research.md).

The five rules

1. One hero per screen. Every screen answers one question first; everything else supports it.
2. Speed is the feature. Repeat logging in as few taps as possible: pre-filled values, last-session numbers inline, timers that start themselves.
3. Trend over noise. Charts show the smooth line over the pale raw data, with a caption that says fluctuation is normal.
4. Never shame. Red is for errors and deletions only — never for a missed target or an over-target meal.
5. Lime is brand, not wallpaper. The accent marks the action and the trend; the rest of the screen stays calm.
